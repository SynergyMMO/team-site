================================================================================
POKEDEX PAGE FEATURES
================================================================================

1. MODE SELECTOR
   Switch between two tracking modes to suit your collection needs.
   - Shiny Mode: Displays caught shinies and fills remaining evolution lines
   - Living Dex Mode: Requires every Pokédex ID obtained in team boxes (no evolution fill)


2. SEARCH BAR
   Real-time search functionality for quick Pokémon lookup.
   - Case-insensitive searching
   - Supports partial name matching (e.g., "chu" finds "pikachu")
   - Clears automatically when applying other filters


3. ENCOUNTER TYPE FILTER
   Filter Pokémon by how common they are to find in the wild.
   - Very Common: Easily found Pokémon with high frequency
   - Common: Standard encounters
   - Uncommon: Less frequent encounters
   - Rare: Difficult to find
   - Very Rare: Extremely scarce encounters
   - Fishing: Only found by fishing with rods
   - Horde: Found exclusively in multi-Pokémon horde battles
   - Lure: Specialized lure-based encounters


4. TIER (RARITY) FILTER
   Filter by overall rarity tier classification (0-7).
   - Tier 0: Very common (Pidgey, Rattata, etc.)
   - Tier 1-3: Common to uncommon range
   - Tier 4-5: Rare Pokémon
   - Tier 6: Very rare encounters
   - Tier 7: Legendary and mythical Pokémon


5. EGG GROUP FILTER
   Filter Pokémon by breeding compatibility group.
   - Supports all egg groups (Dragon, Mineral, Flying, Field, etc.)
   - Reports compatibility for breeding purposes
   - Useful for breeding strategy planning


6. TYPE FILTER
   Filter by Pokémon type classification.
   - Supports single and multiple type selections
   - Select Fire, Water, Electric, Grass, and 14+ other types
   - Multiple selections use AND logic (all selected types required)


7. MOVE FILTER
   Search for Pokémon by moves they can learn.
   - Supports up to 4 simultaneous move searches
   - Searches against learnable moves database
   - Partial move name matching supported
   - Finds level-up, TM/HM, egg, and tutor moves


8. ABILITY FILTER
   Search for Pokémon by ability name.
   - Searches both regular and hidden abilities
   - Partial ability name matching supported
   - Case-insensitive search functionality


9. LOCATION FILTER
   Filter Pokémon by specific in-game location.
   - Requires both location name and region (e.g., "ROUTE 3 - KANTO")
   - Auto-complete suggestions for quick selection
   - Auto-populates when clicking location cards from Pokémon Detail page


10. STAT MINIMUMS FILTER
    Filter Pokémon by minimum base stat values.
    - Set minimum thresholds for HP, ATK, DEF, SP.ATK, SP.DEF, SPD
    - Color-coded stats: Red (low), Orange (below avg), Yellow (average), Green (high)
    - Leave fields blank to ignore specific stats


11. FILTER COMBINATION
    All filters work together simultaneously for precise searching.
    - Combine type, location, rarity, and stats in one query
    - Legendaries automatically hidden unless specifically selected
    - Active filter count displayed at top


12. HIDE COMPLETE TOGGLE
    Hide already-collected Pokémon from results.
    - Available in Synergy Data mode only
    - Focuses results on remaining captures needed


13. SYNERGY DATA TOGGLE
    Display only Pokémon relevant to team collection progress.
    - Shows which team members have caught each Pokémon
    - Displays completion status across team
    - Hides unobtainable legendary data


14. RESULTS GRID DISPLAY
    Visual card-based layout for browsing Pokémon.
    - GIF/image animation display
    - Pokémon name, national ID, and tier badge
    - Type badges for type identification
    - Owner badges showing which team members caught it
    - Clickable cards link to detailed Pokémon page


15. MOBILE RESPONSIVENESS
    Fully optimized mobile interface with adaptive design.
    - Hamburger menu (☰) expands/collapses filter panel
    - 2-column dropdown layout for compact viewing
    - Automatic menu closure when resizing to desktop
    - Touch-friendly filter interactions
    - Portrait orientation optimized


16. FILTER SUMMARY DISPLAY
    Quick view of all active filters and search parameters.
    - Shows count of active filters ("5 Filters Applied")
    - Clickable to modify individual filters
    - Real-time update as filters change


================================================================================
POKEMON DETAIL PAGE FEATURES
================================================================================

1. PAGE HEADER
   Navigation and identification information at top of page.
   - Pokémon name displayed prominently
   - National Pokédex ID number (e.g., #025)
   - "Unobtainable" badge for unavailable Pokémon
   - Back button to previous page
   - Left/right navigation arrows to browse adjacent ID numbers


2. BASIC INFORMATION CARD
   Core Pokémon statistics and metadata overview.
   - Official artwork display
   - Height in meters
   - Weight in kilograms
   - Gender ratio with visual bar chart (male/female/genderless)
   - Generation classification


3. TYPE AND ABILITY SECTION
   Type classifications and move-affecting abilities.
   - Primary and secondary type badges with colors
   - Normal ability display
   - Hidden ability indicator (✨ symbol)
   - Interactive ability hover tooltips showing detailed descriptions
   - Battle effect explanations for each ability


4. BASE STATS CHART
   Visual comparison of stat distribution across six categories.
   - Hit Points (HP)
   - Attack (ATK)
   - Defense (DEF)
   - Special Attack (SP.ATK)
   - Special Defense (SP.DEF)
   - Speed (SPD)
   - Color-coded bars: Red (low/weak), Yellow (average), Green (high/strong)
   - Numerical values displayed alongside bars
   - Max stat cap at 200 for scaling


5. EVOLUTION LINE
   Complete evolution chain with progression details.
   - Base form at top of chain
   - Directional arrows showing evolution flow
   - Evolution condition descriptions (level-up, items, trades, special conditions)
   - Support for branching evolutions with Option A/B labels
   - Clickable Pokémon names to view alternate forms
   - Hover highlight effects for interactivity


6. EGG GROUP AND BREEDING INFORMATION
   Breeding compatibility and catch difficulty data.
   - Egg group classifications
   - Determines compatible breeding partners
   - Horde indicators for mass battle encounters


7. CATCH RATE CALCULATOR
   Interactive hover tooltip with catching probability analysis.
   - Base catch rate display (1-255 scale)
   - Color-coded difficulty: Red (hard), Orange (moderate), Yellow (easy), Green (very easy)
   - Pokéball effectiveness comparison (Pokéball, Great Ball, Ultra Ball, Dusk Ball, Timer Ball, Safari Ball)
   - HP condition analysis (100% HP, lowered HP, asleep status)
   - Best method recommendation
   - Turn count and formula calculations


8. LOCATION INFORMATION
   Complete encounter location database with filtering.
   - Location name and region
   - Level range for encounters
   - Rarity classification
   - Time availability (Morning, Day, Night, All)
   - Habitat/encounter type (Grass, Water, Fishing, Horde, Headbutt, etc.)
   - Clickable location cards auto-populate Pokédex location filter
   - Dynamic encounter type display


9. LEARNABLE MOVES
   Comprehensive moveset with learning method organization.
   - Organized by learning method: Level-up, TM/HM, Tutors, Egg moves
   - Sorted by level for natural progression
   - Move name, type badge, category (Physical/Special/Status)
   - Power value (base damage)
   - Accuracy percentage
   - Level at which move is learned
   - Learning method display (Lvl X, TM, Egg Move, Move Tutor, etc.)


10. SHINY STATUS DISPLAY
    Team collection tracking for shiny variant.
    - Shows which team members have caught shiny form
    - Displays count per player
    - Player names/avatars for identification
    - Completion indicators for team-wide collection
