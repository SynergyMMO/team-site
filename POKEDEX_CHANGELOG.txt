POKEDEX PAGE - NEW FILTERS & FUNCTIONALITY
-------------------------------------------

1. MODE SELECTOR - Shiny vs Living Dex Tracking
   Not explicitly mentioned but core feature. Allows switching between:
   - Shiny Mode: Track caught shinies with evolution line completion
   - Living Dex Mode: Track individual Pokédex IDs without evolution fill

2. TYPE FILTER
   Filter Pokémon by elemental type (Fire, Water, Electric, Grass, etc.)
   Supports multiple simultaneous type selections

3. EGG GROUP FILTER
   Filter by breeding compatibility group (Dragon, Mineral, Flying, etc.)
   Goes beyond just listing egg groups - actively filters results

4. STAT MINIMUMS FILTER
   Set minimum threshold values for any of 6 stats (HP, ATK, DEF, SP.ATK, SP.DEF, SPD)
   Color-coded stat visualization for reference

5. ABILITY FILTER
   Search for Pokémon by ability name (regular or hidden)
   Supports partial name matching

6. FILTER COMBINATION SYSTEM
   All filters work together simultaneously
   Automatically hides legendaries unless specifically selected
   Displays count of active filters

7. HIDE COMPLETE TOGGLE
   Hide already-collected Pokémon from results (Synergy Data mode)
   Focuses results on remaining captures needed

8. SYNERGY DATA TOGGLE
   Display team-specific collection tracking
   Shows which team members have caught each Pokémon
   Filters to only relevant team data

9. RESULTS GRID DISPLAY
   Visual card layout with animated GIFs
   Shows Pokédex ID, tier badge, type badges, owner badges
   Direct click-through to detail pages

10. MOBILE RESPONSIVENESS
    Hamburger menu (☰) for compact filter access
    2-column dropdown layout for mobile
    Touch-friendly interface
    Auto-responsive design


POKEMON DETAIL PAGE - NEW FEATURES
-----------------------------------

1. ABILITY HOVER TOOLTIPS
   Interactive tooltip system showing detailed ability descriptions
   Explains battle effects and interactions
   Distinguishes regular vs hidden abilities

2. TYPE EFFECTIVENESS CHART
   Display type weaknesses, resistances, and immunities
   Shows what types are strong/weak against this Pokémon
   Helpful for battle strategy

3. EVOLUTION LINE VISUALIZATION
   Complete evolution chain with directional arrows
   Evolution condition descriptions (level-up, items, trades, special)
   Support for branching evolutions (Option A/Option B)
   Clickable Pokémon names within chain

4. NAVIGATION ARROWS
   Previous/Next buttons to browse adjacent Pokédex numbers
   Quick access to neighboring Pokémon without returning to Pokédex

5. DETAILED CATCH RATE CALCULATOR
   Interactive hover tooltip with catch probability analysis
   Compares effectiveness of different Pokéballs (Pokéball, Great Ball, Ultra Ball, etc.)
   Analyzes HP conditions and sleep status
   Recommends optimal catching method
   Includes formula and turn count calculations

6. GENDER RATIO VISUALIZATION
   Visual bar chart showing male/female distribution
   Displays "Genderless" designation when applicable

7. GENERATION CLASSIFICATION
   Shows which generation Pokémon originates from

8. LOCATION AUTO-FILTER LINKING
   Click location card on Detail page → auto-filters Pokédex to that location
   Seamless cross-page navigation

9. ENCOUNTER TYPE BREAKDOWN
   Detailed encounter type display (Singles, Horde, Fishing, Headbutt, etc.)
   Shows rarity per location
   Lists time availability (Morning, Day, Night)


SYSTEM FEATURES
---------------

1. KEYBOARD SHORTCUTS
   Ctrl+F / Cmd+F: Browser search in visible results
   Backspace: Back navigation

2. CROSS-PAGE NAVIGATION
   Direct URL linking: /pokemon/{pokemon-name}
   Auto-population of filters between pages
   Clickable evolution chains
   Location card integration

3. DATA CACHING
   First load fetches complete database
   Cached data for faster subsequent interactions
   Local image GIF caching

4. FILTER SUMMARY DISPLAY
   Shows active filter count ("5 Filters Applied")
   Clickable to modify individual filters

5. AUTO-COMPLETE FOR LOCATIONS
   Location filter suggestions dropdown
   Quick selection without typing full entry

6. PARTIAL NAME MATCHING
   Search and filters support partial matching
   Case-insensitive across all search functions
